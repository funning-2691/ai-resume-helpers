## 1. 产品简介

AI简历助手是一个基于大模型的简历优化工具，帮助用户：
- 针对目标岗位优化简历内容
- 评估简历与岗位的匹配度
- 保存历史优化记录

**访问地址**：https://ai-resume-helpers.vercel.app
> 注：国内访问可能需要网络代理工具。

---

## 2. 快速上手（首次使用）

| 步骤 | 操作 | 说明 |
| :--- | :--- | :--- |
| 1 | 打开访问地址 | 进入工具首页 |
| 2 | 准备岗位描述和简历 | 可以是文本或文件（.txt/.docx） |
| 3 | 填写/上传内容 | 参考第3节详细说明 |
| 4 | 点击“开始优化” | 等待10-30秒获取结果 |

## 3. 核心任务

### 3.1 如何优化简历

**前置条件**：已准备好岗位描述（JD）和个人简历的文本或文件。

**操作步骤**：

| 步骤 | 动作 | 说明 |
| :--- | :--- | :--- |
| 1 | 输入岗位描述 | 粘贴到页面左上方文本框，或点击“上传JD”按钮上传文件 |
| 2 | 输入个人简历 | 粘贴到页面左下方文本框，或点击“上传简历”按钮上传文件 |
| 3 | 开始优化 | 点击页面右上角 **“开始优化”** 按钮 |
| 4 | 等待处理 | 系统显示“处理中”加载状态，约10-30秒 |
| 5 | 查看结果 | 右侧区域展示匹配度评分和优化后的简历内容 |

**预期结果**：页面右侧显示0-100%的匹配度分数，以及AI优化后的简历文本。

### 3.2 如何保存优化结果

**操作步骤**：
1. 在优化结果区域，点击 **“复制”** 按钮 → 可粘贴到Word/其他编辑器
2. 点击 **“下载”** 按钮 → 选择 **TXT** 或 **Word** 格式保存到本地

### 3.3 如何查看历史记录

**操作步骤**：
1. 点击页面右上角 **“历史记录”** 按钮
2. 系统展示历史优化列表（按时间倒序）
3. 点击任意一条记录，可查看完整内容

**说明**：历史记录存储在本地数据库中。清除浏览器缓存会导致记录丢失，建议重要记录手动导出保存。

## 4. 常见问题（FAQ）

| 问题 | 解答 |
| :--- | :--- |
| 优化结果不满意怎么办？ | 可修改原文后再次点击优化。由于AI的随机性，每次结果可能略有不同，多试几次可获得满意版本。 |
| 上传文件失败？ | 当前支持 **.txt** 和 **.docx** 格式，文件大小建议不超过5MB。如仍失败，请尝试直接粘贴文本。 |
| 匹配度评估怎么用？ | 填写岗位描述和简历后，点击“开始优化”，结果区域会自动展示匹配度百分比。 |
| 历史记录能保存多久？ | 目前为本地永久保存，直到浏览器缓存被清除。后续计划升级为云同步功能。 |
| 国内无法访问怎么办？ | 该地址部署在Vercel平台，国内部分网络环境受限。建议使用网络代理工具，或等待后续国内部署方案。 |
